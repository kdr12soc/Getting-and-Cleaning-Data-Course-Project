All files were extracted from the zip file UCI HAR Dataset heirarchy.

"subject" was obtained from the files "subject_text.txt" and "subject_train.txt", and were binded to the train and test tables.

"Code" is a variable that shows the names of the differenta activities. It was imputed with the following code line, from the "activity_labels" table: 
Data.mean.std$code <- activity_labels[Data.mean.std$code, 2]

The remaining variables are means and standard deviations of the variables from the original data set. They were extracted from a larger data set containing multiple variables, after selecting only those with "mean" or "std" on their name.
